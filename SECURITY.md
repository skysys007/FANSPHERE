# Security Policy

## Supported Versions

Currently, only the main branch (`latest`) is actively supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| `< 1.0` | :x:                |

## Reporting a Vulnerability

We take the security of FANSPHERE extremely seriously. If you discover a security vulnerability, please do NOT report it by opening a public GitHub issue.

Instead, please send an email to the repository owner or submit a private vulnerability report via GitHub's Security tab.

When reporting a vulnerability, please include:
*   A detailed description of the vulnerability.
*   The exact steps required to reproduce the issue.
*   Your proposed mitigation strategy (if any).

We will triage the vulnerability within 48 hours and release a patch as quickly as possible.
